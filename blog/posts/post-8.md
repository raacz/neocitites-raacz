My theme for 2026 is Wholesomeness, and part of that is trying to do less doomscrolling and more reading. I've read 43 books so far this year, and I'd like to share some notes here on the tools I've been using within my reading workflow. 


# General Overview: 

- Library Management: Calibre
- File transfers: LocalSend
- Social platform: Hardcover
- E-reader: Kobo Aura first edition (N514)
- E-reader software: Koreader with plugins: SimpleUI, LocalSend
- Reading app for phone: Readest (to read books with TTS)
